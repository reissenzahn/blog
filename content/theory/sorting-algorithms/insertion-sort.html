--- title: "Insertion Sort" date: 2021-11-24 draft: false ---

Insertion sort is a in-place sorting algorithm that proceeds by growing a sorted output list one element at a time. At each iteration, one element is removed from the input and inserted at the correct location within the sorted list. Insertion sort performs well on partially sorted arrays and small arrays.

Procedure

Insertion sort proceeds as follows:

  1. Start at the second element of the array.
  2. Compare the current element to the element to its left. If the current element is smaller then swap the elements.
  3. Continuing swapping until the current element is larger than the element to its left or the end of the array is reached.
  4. Repeat this process for each subsequent element of the array.

Implementation

Complexity

Worst case time \(O(n^2)\)
Best case time \(O(n)\)
Average case time \(O(n^2)\)
Space \(O(1)\)

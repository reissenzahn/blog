--- title: "Stack" date: 2021-11-24 draft: false ---
  • A stack is a linear data structure which allows elements to be inserted or removed at one end.
  • The end at which elements are inserted and removed is called the top.
  • A stack is a last in, first out (LIFO) data structure as the last element to be inserted will be the first one to be removed.

Structure

  • A stack can be represented using a singly-linked list with a pointer to the first element in the list being used to keep track of the top.
  • Alternatively, an array-based implementation can be used. The underlying array will need to be resized if the number of elements in the stack exceeds the size of the array.

Operations

push()

  • The push operation inserts an element at the top of the stack.

pop()

  • The pop operation removes the most recently added element that was not yet removed.
  • A non-essential peek operation may be included to return the most recently added element without removing it.

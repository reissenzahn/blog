--- title: "19. Remove Nth Node From End of List" date: 2021-02-22 draft: false ---

Solution

{{% code java %}}class Solution { public ListNode removeNthFromEnd(ListNode head, int n) { ListNode sentinel = new ListNode(-1); sentinel.next = head; ListNode fast = sentinel; ListNode slow = sentinel; // slow will wait n times int i = 0; while (fast.next != null) { if (i == n) { slow = slow.next; } else { i++; } fast = fast.next; } // delete N-th node slow.next = slow.next.next; return sentinel.next; } }{{% /code %}}

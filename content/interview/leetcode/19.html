--- title: "19. Remove Nth Node From End of List" date: 2021-02-22 draft: false ---

Question

Given the head of a linked list, remove the n-th node from the end of the list and return its head.

Usage

{{% code makefile %}}Example 1: Input: head = [1,2,3,4,5], n = 2 Output: [1,2,3,5] Example 2: Input: head = [1], n = 1 Output: [] Example 3: Input: head = [1,2], n = 1 Output: [1]{{% /code %}}

Solution

{{% code java %}}/** * public class ListNode { * int val; * ListNode next; * ListNode() {} * ListNode(int val) { this.val = val; } * ListNode(int val, ListNode next) { this.val = val; this.next = next; } * } */ class Solution { public ListNode removeNthFromEnd(ListNode head, int n) { ListNode sentinel = new ListNode(-1); sentinel.next = head; ListNode fast = sentinel; ListNode slow = sentinel; // slow will wait n times int i = 0; while (fast.next != null) { if (i == n) { slow = slow.next; } else { i++; } fast = fast.next; } // delete N-th node slow.next = slow.next.next; return sentinel.next; } }{{% /code %}}

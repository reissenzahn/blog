Question

Given a sorted array of distinct integers and a target value, return the index if the target is found. If not, return the index where it would be if it were inserted in order.

You must write an algorithm with \(O(\log n)\) runtime complexity.

Usage

{{% code makefile %}}Input: nums = [1,3,5,6], target = 5 Output: 2 Input: nums = [1,3,5,6], target = 2 Output: 1 Input: nums = [1,3,5,6], target = 7 Output: 4 Input: nums = [1,3,5,6], target = 0 Output: 0 Input: nums = [1], target = 0 Output: 0{{% /code %}}

Solution

{{% code java %}}public int searchInsert(int[] nums, int target) { int l = 0; int r = nums.length - 1; while (l <= r) { int m = (l + r) / 2; if (nums[m] < target) { l = m + 1; } else if (nums[m] > target) { r = m - 1; } else { return m; } } return r + 1; }{{% /code %}}

--- title: "876. Middle of the Linked List" date: 2021-02-22 draft: false ---

Solution

{{% code java %}}class Solution { public ListNode middleNode(ListNode head) { ListNode fast = head; ListNode slow = head; while (fast != null && fast.next != null) { slow = slow.next; fast = fast.next.next; } return slow; } }{{% /code %}}

Complexity

O(n) time and O(1) space.

--- title: "238. Product of Array Except Self" date: 2021-02-22 draft: false ---

Solution

{{% code java %}}class Solution { public int[] productExceptSelf(int[] nums) { int[] answer = new int[nums.length]; int prod1 = 1; for (int i = 0; i < nums.length; i++) { answer[i] = prod1; prod1 *= nums[i]; } int prod2 = 1; for (int i = nums.length - 1; i >= 0; i--) { answer[i] *= prod2; prod2 *= nums[i]; } return answer; } }{{% /code %}}

Complexity

O(n) time and O(1) space.

Introduction

A queue is a collection of elements that are maintained in a sequence and can be modified by the insertion of elements at one end and the removal of elements from the other end. The end of the sequence at which elements are added is called the back and the end at which elements are removed is called the front. A queue is a first in, first out (FIFO) data structure as the first element to be inserted will be the first one to be removed.

Enqueue

The enqueue operation inserts an element at the back of the queue. This can be implemented using a linked list by inserting an element at the tail of the list.

Dequeue

The dequeue operation removes an element from the front of the queue. This can be implemented using a linked list by removing the element at the head of the list.

A non-essential peek operation can also be included to return the value of the next element to be dequeued without dequeuing it.

Implementation

public class Queue {

  private Node front;
  private Node back;

  public void enqueue(T value) {
    Node node = new Node<>(value, null);

    if (back != null)
      back.next = node;
    
    back = node;

    if (front == null)
      front = back;
  }

  public T dequeue() {
    if (isEmpty())
      throw new NoSuchElementException();

    T value = front.value;
    front = front.next;
    
    if (isEmpty())
      back = null;

    return value;
  }

  public T peek() {
    if (isEmpty())
      throw new NoSuchElementException();

    return front.value;
  }
  
  public boolean isEmpty() {
    return front == null;
  }

  private static class Node {

    private T value;
    private Node next;

    public Node(T value, Node next) {
      this.value = value;
      this.next = next;
    }
  }
}

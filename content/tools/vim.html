Concepts

Modes

  • Normal/Command Mode: Used for entering commands.
  • Insert Mode: Used for making changes to the file.
  • Visual Mode: Used for selecting text using movements.
  • Ex Mode: Used for entering commands at the ":" prompt.

Configuration

{{% code vim %}}$ vim ~/.vimrc " remap ESC to jk inoremap jk " remap leader key to ' let mapleader = "'" " enable syntax highlighting syntax on " show line numbers set number " show current filetype set filetype " disable the swapfile set noswapfile " highlight all search results set hlsearch " ignore case in search set ignorecase " show search results as you type set incsearch " enable spellchecking set spell spelllang=en_us{{% /code %}}

Files

{{% code vim %}}" save changes :w " save changes and quit :wq ZZ " quit without saving changes :q! " save file to a location :saveas ~/path/filename.txt " change file type set ft{{% /code %}}

Movement

Basic Movements

{{% code vim %}}" move up one line k " move down one line j " move right one character l " move left one character h{{% /code %}}

Moving on the Current Line

{{% code vim %}}" move to the beginning of the line 0 " move to the end of the line $ " jump to character on line f(character) " jump to just before character on line t(character) " jump backwards to character on line F(character) " jump backwards to just before character on line T(character) " repeat previous jump ; " repeat previous jump in opposite direction ,{{% /code %}}

Moving by Word, Sentence or Paragraph

{{% code vim %}}" move to the end of the current word e " move forward one word w " move back one word b " move forward one word (ignore some delimiters that may break words) W " move back one word (ignore some delimiters that may break words) B " move forward one sentence ) " move forward one paragraph }{{% /code %}}

Moving within the Screen

{{% code vim %}}" move to the top of the screen H " move to the middle of the screen M " move to the bottom of the screen L{{% /code %}}

Moving within the File

{{% code vim %}}" move to a given line number :line_number " move to the top of the file gg " move to the bottom of the file G " move up half a screen ^U " move down half a screen ^D " move down a page ^F " move up a page ^B{{% /code %}}

Changing Text

{{% code vim %}}" insert before the cursor i " insert at beginning of line I " append after the cursor a " append at the end of the line A " open a new line below the current line o " open a new line above the current line O " replace the character under the cursor r " replace the character under the cursor but keep typing afterwards R {{% /code %}}

Deleting Text

{{% code vim %}}" delete character under the cursor x " delete the character before the cursor X " delete the current line dd " delete to end of the line D " join the current line with the next one (delete whatever is between) J " delete movement 'm' (e.g. word, sentence, paragraph) dm {{% /code %}}

Visual Mode

{{% code vim %}}" enter character-based visual mode v " enter line-based visual mode V " enter paragraph-based visual mode CTRL-v " select inside of parenthesis, brackets, etc. vi( vi[ {{% /code %}}

Undo, Redo and Repeat

{{% code vim %}}" undo your last action u # redo the last action CTRL-r # repeat last action . " repeat last 5 actions 5.{{% /code %}}

Copy and Paste

{{% code vim %}}" copy selection y " copy current line yy " paste after the current cursor position p " paste before the current cursor position P{{% /code %}}

Searching

{{% code vim %}}" search for string /(query) " search backwards for string ?query " go to next search result n " go to previous search result N " search for word under the cursor * " clear search highlights :noh {{% /code %}}

Substitution

{{% code vim %}}" change 'foo' to 'bar' on every line :%s /foo/bar/g " change 'foo' to 'bar' on the current line :s /foo/bar/g{{% /code %}}

Macros

{{% code vim %}}" start recording a macro named 'a' qa " stop recording q " play macro named 'a' @a{{% /code %}}

Spellchecking

{{% code vim %}}" enable spellchecking :set spell " disable spellchecking :set nospell " go to next misspelled word ]s " go to previous misspelled word [s " get suggestions for current misspelled word z= " mark a misspelled word as correct zg " mark a word as misspelled zw{{% /code %}}

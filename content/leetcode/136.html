--- title: "136. Single Number" date: 2021-02-22 draft: false ---

Solution

{{% code java %}}class Solution { public int singleNumber(int[] nums) { // a ^ a = 0 // ^ is commutative // a ^ 0 = a int res = 0; for (int n : nums) { res ^= n; } return res; } }{{% /code %}}

Complexity

O(n) time and O(1) space.

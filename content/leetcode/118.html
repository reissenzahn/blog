--- title: "118. Pascal's Triangle" date: 2021-02-22 draft: false ---

Question

Given an integer numRows, return the first numRows of Pascal's triangle.

Usage

{{% code makefile %}}Example 1: Input: numRows = 5 Output: [[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]] Example 2: Input: numRows = 1 Output: [[1]]{{% /code %}}

Solution

We populate a list of lists with the following structure: 

  0 1 2 3 4
  0 1
  1 1 1
  2 1 2 1
  3 1 3 3 1
  4 1 4 6 4 1
{{% code java %}}class Solution { public List> generate(int numRows) { List> result = new ArrayList<>(); // add [1] as the first row List firstRow = new ArrayList<>(); firstRow.add(0, 1); result.add(firstRow); // populate remaining rows for (int i = 1; i < numRows; i++) { List row = new ArrayList<>(); row.add(1); for (int j = 1; j < i; j++) { List prevRow = result.get(i - 1); // above + one to the left row.add(prevRow.get(j) + prevRow.get(j - 1)); } row.add(1); result.add(row); } return result; } }{{% /code %}}

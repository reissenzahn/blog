--- title: "21. Merge Two Sorted Lists" date: 2021-02-22 draft: false ---

Question

Merge two sorted linked lists and return it as a sorted list. The list should be made by splicing together the nodes of the first two lists.

Usage

{{% code makefile %}}Input: l1 = [1,2,4], l2 = [1,3,4] Output: [1,1,2,3,4,4] Input: l1 = [], l2 = [] Output: [] Input: l1 = [], l2 = [0] Output: [0]{{% /code %}}

Solution

{{% code java %}}public ListNode mergeTwoLists(ListNode l1, ListNode l2) { if (l1 == null) return l2; if (l2 == null) return l1; ListNode l; if (l1.val < l2.val) { l = l1; l.next = mergeTwoLists(l1.next, l2); } else { l = l2; l.next = mergeTwoLists(l1, l2.next); } return l; }{{% /code %}}

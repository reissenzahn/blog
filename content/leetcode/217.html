Solution

{{% code java %}}class Solution { // O(n log n) time; O(1) space // public boolean containsDuplicate(int[] nums) { // Arrays.sort(nums); // for (int i = 0; i < nums.length - 1; i++) { // if (nums[i] == nums[i + 1]) { // return true; // } // } // return false; // } // O(n) time; O(n) space public boolean containsDuplicate(int[] nums) { Set set = new HashSet(); for (int n : nums) { if (set.contains(n)) { return true; } set.add(n); } return false; } }{{% /code %}}

Complexity

The sorting approach is O(n log n) time and O(1) space.

The set approach is O(n) time and O(n) space.

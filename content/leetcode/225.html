--- title: "225. Implement Stack using Queues" date: 2021-02-22 draft: false ---

Question

Implement a last-in-first-out (LIFO) stack using only two queues. The implemented stack should support all the functions of a normal stack (push, top, pop, and empty).

Usage

{{% code makefile %}}Example 1: MyStack myStack = new MyStack(); myStack.push(1); myStack.push(2); myStack.top(); // return 2 myStack.pop(); // return 2 myStack.empty(); // return False{{% /code %}}

Solution

Popping an element is achieved by simply dequeueing and enqueuing all the elements in the queue except the last one which is dequeued and returned. There is no way implement the pop operation in less than O(n) time.

{{% code java %}}class MyStack { private Queue queue; public MyStack() { this.queue = new LinkedList<>(); } public void push(int x) { this.queue.offer(x); } public int pop() { for (int i = 0; i < this.queue.size() - 1; i++) { int val = this.queue.poll(); this.queue.offer(val); } return queue.poll(); } public int top() { int result = this.pop(); this.push(result); return result; } public boolean empty() { return this.queue.isEmpty(); } }{{% /code %}}

Solution

Suppose we have n coins and the answer is k. Then completed rows contain 1 + 2 + ... + k = k(k+1)/2 coins.

We can then reformulate the problem as finding the maximum k such that k(k+1)/2 â‰¤ n. This can be solved using binary search on the array [0, ..., n].

{{% code java %}}class Solution { public int arrangeCoins(int n) { long lo = 0; long hi = n; while (lo <= hi) { long k = (lo + hi) / 2; long numCoins = k * (k + 1) / 2; if (numCoins == n) { return (int) k; } else if (numCoins < n) { lo = k + 1; } else { hi = k - 1; } } return (int) hi; } }{{% /code %}}

Introduction

A priority queue is a queue-like data structure in which elements are enqueued with an associated priority. The dequeue operation removes and returns the element from the queue that has the highest priority. A min-priority queue considers lower values to have higher priority, while a max-priority queue considers higher values to have higher priority. For performance reasons, priority queues are frequently implemented using a heap.

Enqueue

For a min-priority queue implemented with a min-heap, the enqueue operation can be achieved by inserting the element into the heap using its priority.

Dequeue

For a min-priority queue implemented with a min-heap, the dequeue operation can be achieved by removing the minimum element in the heap.

Complexity

A priority queue implemented with a binary heap has the following time complexities:

Operation Worst-case Complexity
Peek \(O(1)\)
Enqueue \(O(\log n)\)
Dequeue \(O(\log n)\)

--- title: "451. Sort Characters By Frequency" date: 2020-09-29 draft: false ---

Question

Given a string, sort it in decreasing order based on the frequency of characters.

Usage

Input:
"tree"

Output:
"eert"

Explanation:
'e' appears twice while 'r' and 't' both appear once.
So 'e' must appear before both 'r' and 't'. Therefore "eetr" is also a valid answer.
Input:
"cccaaa"

Output:
"cccaaa"

Explanation:
Both 'c' and 'a' appear three times, so "aaaccc" is also a valid answer.
Note that "cacaca" is incorrect, as the same characters must be together.
Input:
"Aabb"

Output:
"bbAa"

Explanation:
"bbaA" is also a valid answer, but "Aabb" is incorrect.
Note that 'A' and 'a' are treated as two different characters.

Solution

  • Add the characters to a map with their frequencies as the associated values.
  • Add those entries to a max-heap ordered on their values. 
public String frequencySort(String s) {
  Map map = new HashMap<>();

  // add characters and their frequencies to map
  for (char c : s.toCharArray())
    map.put(c, map.getOrDefault(c, 0) + 1);
  
  // provide comparator to use a max-heap based on frequency
  PriorityQueue<Map.Entry> pq = new PriorityQueue<>((a, b) -> b.getValue() - a.getValue());

  // add map entries to heap
  pq.addAll(map.entrySet());

  StringBuilder sb = new StringBuilder();

  // build up output from heap contents
  while (!pq.isEmpty()) {
    Map.Entry e = pq.poll();

    // use the frequency to determine how many values
    for (int i = 0; i < (int) e.getValue(); i++) {
      sb.append(e.getKey());
    }
  }

  return sb.toString();
}

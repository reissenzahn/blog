Introduction

Breadth-first search is a graph traversal algorithm. Given a graph and a source vertex, it explores the graph to discover every vertex reachable from the source vertex. The algorithm cautiously explores the graph in layers starting from the first layer which consists of the source vertex. Subsequent layers consist of those vertices that are the neighbors of the previous layer's vertices that do not appear in any earlier layer.

Procedure

To traverse a graph, breadth-first search proceeds as follows:

  1. Initialize an empty queue.
  2. Mark the source vertex as visited and enqueue it.
  3. Dequeue a vertex and consider each of its adjacent vertices.
  4. For each adjacent vertex, if the vertex has not been visited then mark it as visited and enqueue it.
  5. Repeat steps 3-4 until the queue is empty.

Example

Complexity

Breath-first search has worst-case performance of \(O(|V| + |E|)\) and worst-case space complexity of \(O(|V|)\).

Implementation

public class BFS {

  public static void traverse(List> g, int start) {
    boolean[] visited = new boolean[g.size()];
    Queue queue = new LinkedList<>();

    visited[start] = true;
    queue.offer(start);

    int count = 0;

    System.out.print("traversal order: ");

    while (!queue.isEmpty()) {
      int v = queue.poll();
      count++;
      System.out.print(v + " ");
      for (Integer w : g.get(v)) {
        if (!visited[w]) {
          visited[w] = true;
          queue.offer(w);
        }
      }
    }

    System.out.println("\ntotal vertices: " + count);
  }
}

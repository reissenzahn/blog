--- title: "Cycle Detection" date: 2021-11-24 draft: false ---

A cycle in a graph is a non-empty sequence of edges in which only the first ad last vertices are equal. The existence of a cycle in directed and undirected graphs can be determined through applications of depth-first search. Many topological sorting algorithms also detect cycles.

Directed Graphs

We start by declaring the sets white, gray and black. These sets are defined as follows:

  • White: vertices that have not been visited at all (initially contains all vertices in the graph).
  • Gray: vertices currently being visited.
  • Black: vertices that have been "totally" visited (visited and have had all their children visited).

We proceed by iterating over each of the vertices in the white set in arbitrary order. For each vertex, we begin a depth-first search starting at the vertex. The depth-first search proceeds as follows:

  1. Move the start vertex from the white set to the graph set.
  2. Consider each of the adjacent vertices
